# Archivierte Referenzen

Dieser Bereich enthält historische Referenzmaterialien aus früheren OpenClaw-/Nexus-Setups.

`docs/archiv/referenzen/setup/` ist ausschließlich Archivmaterial. Der Inhalt wird nicht importiert, nicht in Tests als Pflichtbestand verlangt, nicht von Runtime-Code gelesen und nicht als Source of Truth behandelt.

Die aktive Source of Truth bleibt `nexus/*.yml` und später `nexus.db`.
